Laço

Os laços (ou loops) em programação são estruturas fundamentais que permitem a execução repetida de um bloco de código enquanto uma condição específica é verdadeira. No JavaScript, existem vários tipos de laços, cada um adequado para diferentes situações. Vamos explorar os principais tipos de laços disponíveis em JavaScript: for, while, do...while, e os mais modernos for...of e for...in.

1. Laço for

O laço for é provavelmente o mais conhecido e amplamente utilizado. Ele é ideal quando se sabe antecipadamente quantas vezes se deseja repetir um bloco de código.

2.Laço while

O laço while é usado quando se deseja repetir um bloco de código enquanto uma condição específica for verdadeira. Ele é ideal quando o número de iterações não é conhecido antecipadamente.

3. Laço do...while

O laço do...while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez, pois a condição é avaliada após a execução do bloco.

4. Laço for...of

Introduzido no ECMAScript 6 (ES6), o laço for...of é usado para iterar sobre valores de objetos iteráveis (como arrays, strings, Map, Set, etc.).

5. Laço for...in

O laço for...in é usado para iterar sobre as propriedades enumeráveis de um objeto. Ele é mais adequado para objetos, não para arrays.

Ou seja:

Use for quando souber o número exato de iterações. Use while quando a condição de término depende de algo que pode não ser conhecido antecipadamente. Use do...while quando precisar garantir que o bloco de código seja executado pelo menos uma vez. Use for...of para iterar sobre elementos de coleções iteráveis. Use for...in para iterar sobre propriedades de objetos.
